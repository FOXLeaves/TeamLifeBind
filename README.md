# Team LifeBind
A Minecraft plugin where players share lives with their team.  If one player dies, the entire team dies. Teams spawn randomly and race to complete the game. PvP is enabled.

Minecraft团队生存游戏，全队共用一个血条。在原版世界中随机出生并抢在其他队伍之前通关游戏。
